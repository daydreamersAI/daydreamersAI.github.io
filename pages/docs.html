API Reference

POST /api/v1/generate

Generate responses using our fine-tuned models

Parameters:

  • prompt (string): Input text prompt
  • max_tokens (integer): Maximum response length
  • temperature (float): Response randomness (0-1)
POST /api/v1/train

Fine-tune models on custom data

Parameters:

  • training_data (array): Array of training examples
  • epochs (integer): Number of training epochs
  • model_type (string): Base model to fine-tune
